S14Clock Firmware Installer

Instructions

  • Connect clock via usb-c to computer
  • Select formware and hit the blue "Connect" button below, a pop-up window will appear
  • On the clock side, press and hold '0', then press 'RST', release 'RST', then finally release '0'
  • Select "ESP32-S2 (COM?) Paired" entry and then hit the greem "Connect" button
  • Select the "Install ....Firmware..." option, confirm by clicking "Install"
  • Popup shows progress on Prepare Installation, Erasing, Installing,.., this takes 3 or 4 minutes in total

See the S14Clock Project.

If this ESP32 Web Tools fails to work properly, please try the AdaFruit one, it works better.
